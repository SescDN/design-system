# Starter Kit SESC DN

## Iniciar projeto com install para instalar todas as dependências
```
yarn install
```

### Compilar o código
```
yarn serve
```

### Compilar e minificar os arquivos para dar Build
```
yarn build
```

### Lints e corrigir código
```
yarn lint
```
